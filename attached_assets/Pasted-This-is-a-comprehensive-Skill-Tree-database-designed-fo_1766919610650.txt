How to Implement "Evolution" in Your Code
To make the labels change automatically as you grow, you can add a simple logic check in your JavaScript.

Example Logic:

If INT > 50 AND Engineering_XP > 100 → Change Class to "Electrical Engineer".

If STR > 100 AND Pushups > 5000 → Change Title to "Iron Body".

Code Snippet to Add to Your System:

JavaScript

function checkEvolution() {
    const int = playerStats.INT;
    const str = playerStats.STR;
    
    let jobTitle = "Student"; // Default

    // Engineering Path
    if (int > 20) jobTitle = "Novice Tinkerer";
    if (int > 50) jobTitle = "Apprentice Technician";
    if (int > 100 && playerStats.SEN > 80) jobTitle = "Electrical Engineer";
    if (int > 300) jobTitle = "Systems Architect";

    // Update UI
    document.getElementById('jobTitle').innerText = jobTitle;
}
